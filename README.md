# MaestroVPN TV

Own-brand Android TV VPN client + provisioning/billing backend.
Subscription with 4 switchable protocols (Hysteria2 / VLESS / Naive / AnyTLS),
auto-key (no manual key handling), auto-start on boot, per-app split tunnel,
in-app purchase & renewal (СБП + Telegram approve).

See ARCHITECTURE.md for the full design, infra study, and build order.
